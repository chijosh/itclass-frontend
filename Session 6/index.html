There are several data types in JavaScript: Primitive and Composite data types.

Primites data types: String, boolean, number, null, bigInt, undefined, Symbol.

Composite data types: Object ( example: {name: "Josh"} ), Array ( example: [1,2,3] ), function ( example: function greetings(){console.log("Hello")} )

You can access data in JavaScript Objects in two ways: Dot notation, Bracket notation.
